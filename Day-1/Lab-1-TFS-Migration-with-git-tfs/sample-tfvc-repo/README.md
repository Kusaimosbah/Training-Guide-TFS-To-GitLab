# Legacy Application

This is a sample application migrated from TFS/TFVC to Git/GitLab.

## Features
- Basic calculator operations (Add, Subtract, Multiply, Divide)
- Multiple commits demonstrating TFS changeset history preservation
- Example of real-world TFS-to-GitLab migration workflow

## Migration History
This repository simulates a typical TFVC repository with multiple changesets:
- Changeset 1001: Initial project setup
- Changeset 1002: Add Calculator class
- Changeset 1003: Update Program to use Calculator
- Changeset 1004: Add documentation

## Running the Application
```bash
dotnet run
```
